-- i. S-A Structure
--- 1. State
A state is a set of representations, describes the momentary feeling of one individual.
---- a) Representation
Representation is a set of Representation Units.
Ex. Say a, b, c are 3 Representation Units, then the discrete topology of {a,b,c} will be a subset of the Representations that can be induced from them.
---- b) Representation Unit
The representation unit is the minimal element that constructs a representation. The basic form can be a sensation signal. A representation itself can be a representation unit.
Ex. If we have sensation signals a,b,c, {a, {a,b,c}} is a valid representation.
---- c) Goal
A representation R is a Goal if it's a destination in the graph. A graph can have multiple destinations.
----- i) Constant goal
The goals in Constant Goal Graph are static.
Comment: the Constant Goal Graph was initially used to explain an organism's needs. Thus the Constant Goals are considered as states that need are satisfied.
--- 2. Action
An Action is a path of the S-A Graph, i.e., a series of Action and State. It reflects one's physical and mental activity.
---- a) Action Unit
An action unit is a minimal element that constitutes an action.
Ex1. To study facial expression, we can set the contraction of a piece of facial muscle as the action unit.
Ex2. Say a1 is an action unit, and s1 is a state. Then a1->s1->a1 and a1 are all actions.
----- i) Single thread action
A Single Thread Action indicates that it activates only one action unit at each moment.
----- ii) Multi thread action
A Multi Thread Action is an action that involves multiple threads of actions at some moment, i.e., when performing the action, there exists a time s.t. there is more than one action unit within the compound action activated.
----- iii) Physical action unit
Physical action units are the action unit (simple neural activities) that generate a physiological change (general speaking).
Ex. A muscle contraction, but not a neuron's activation.
----- iv) Cognitive action unit
A Cognitive action unit is the action unit that generates a simple cognitive process.
---- b) Repeat action?
Comment: Since neural circuits generate cognitive processes, it would be hard to find the smallest circuit.
The action unit can be defined by the model designer when a model is set up. However, to give a sense of it, we can consider a cognitive action unit as the smallest possible neural circuit that has a cognitive function.

---- b) Non-goal S-A graph
A non-goal S-A graph is an S-A graph without a goal. It still has destinations of paths, but these only represent the last state perceived after a series of actions.
-- ii. R-R Structure
An R-R graph is a graph of representations s.t. for two connected representations, we can always access one from another.
Comment: While S-A graphs with a single direction can represent procedural, episodic memory, and some types of semantic memory, certain types of semantic memory can not be represented by single direction graphs. Thus we use non-directed.
--- 1) Non-directed graph
A non-directed graph is a graph s.t. for any two connected representations, the edge between them contains nothing. This is the general form of R-R structure and by applying it, we can integrate the S-A structure with the traditional associative network. Two possible concepts can be represented using the R-R structure:
---- a) Priming
---- b) Mirror Neurons
-- iii. Unit Storage
By definition, representations can be generated by representations and actions can be generated by actions. This indicates that, for one representation or action, there normally is a set of representations or actions related to it. There are two methodologies to describe such a relationship. One is through connectionism: R-R structure for representation. Another one is to create a space that records the connection between finer representation/action and coarser representation/action.
Comment: This is not an important concept since it would not affect how the model work, it just provides an idea of how to implement the model.
